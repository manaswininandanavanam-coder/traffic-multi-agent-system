# Multi-Agent Traffic Management System

## Project Overview
This project implements a **Multi-Agent System for Traffic Management** using Python and Streamlit.  
The system simulates intelligent traffic signals that dynamically adjust based on vehicle density.

## Features
- Multi-agent traffic signal control
- Traffic density simulation
- Interactive dashboard
- Traffic analytics charts
- Map-based traffic visualization

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib
- Folium

## Installation

1. Clone the repository

git clone https://github.com/yourusername/traffic-multi-agent-system.git

2. Install dependencies

pip install -r requirements.txt

3. Run the application

streamlit run app.py

## System Workflow
1. Traffic data is generated or loaded from CSV.
2. Traffic agents analyze vehicle density.
3. Agents communicate with neighboring intersections.
4. Signal controller updates traffic lights.
5. Dashboard displays traffic data and maps.
